Exploring csv files#

In this notebook we use bob to explore and plot some tabular data.

from bia_bob import bob

In this example we use a videogame sales dataset by Gregory Smith which is licensed CC-BY 4.0.

%%bob
write python code that
* opens the file vgsales.csv using pandas
* if the file doesn't exist, open it from https://zenodo.org/records/5898311/files/vgsales.csv?download=1 
  and save it as vgsales.csv

To open the file vgsales.csv using pandas and download it from the URL https://zenodo.org/records/5898311/files/vgsales.csv?download=1 if it doesn’t exist, you can use the following code:

import os
import pandas as pd
import urllib.request

# Check if the file exists
if not os.path.exists('vgsales.csv'):
    # Download the file from the URL
    urllib.request.urlretrieve('https://zenodo.org/records/5898311/files/vgsales.csv?download=1', 'vgsales.csv')

# Open the file using pandas
df = pd.read_csv('vgsales.csv')

# Do something with the dataframe
# For example, print the first 5 rows
print(df.head())

%%bob 
group the data by Publisher and 
draw a bar plot of Global_Sales

To group the data by the ‘Publisher’ column and draw a bar plot of the ‘Global_Sales’ column, you can use the following code:

import pandas as pd
import matplotlib.pyplot as plt

# Assuming you already have the data stored in a pandas DataFrame called 'df'

# Group the data by 'Publisher' and calculate the sum of 'Global_Sales'
grouped_data = df.groupby('Publisher')['Global_Sales'].sum()

# Create a bar plot of the grouped data
grouped_data.plot(kind='bar')

# Set the labels and title of the plot
plt.xlabel('Publisher')
plt.ylabel('Global Sales')
plt.title('Global Sales by Publisher')

# Display the plot
plt.show()

import pandas as pd

%%bob
Group the data by Publisher and select the 10 datasets with the highest Global_Sales

To group the data by ‘Publisher’ and select the 10 datasets with the highest ‘Global_Sales’, you can use the following code:

import pandas as pd

# Assuming you already have the data stored in a pandas DataFrame called 'df'

# Group the data by 'Publisher' and calculate the sum of 'Global_Sales'
grouped_data = df.groupby('Publisher')['Global_Sales'].sum()

# Sort the grouped data in descending order and select the top 10 datasets
top_10_data = grouped_data.nlargest(10)

# Print out the top 10 datasets with the highest 'Global_Sales'
print(top_10_data)
